How to play

Use WASD or arrow keys to move. Vampanchino must tag all runners before midnight. Runners win by lasting until midnight or collecting all items. The orange cat darts quickly and sometimes naps.

Share the room code with friends; everyone joins the same code. Host the server separately (see README) and set window.GAME_SERVER to its websocket URL if hosting the frontend elsewhere.
